# Виправлення цін препаратів (0.00 грн ? правильні ціни)

## Проблема
Всі ціни відображаються як **0.00 грн** в каталозі препаратів.

## Причина
В базі даних препарати створені **без цін** (Price = 0).

## Рішення

### Що було зроблено:

1. **Оновлено `Data/DbSeeder.cs`:**
   - Додано 10 українських препаратів з реальними цінами
   - Ціни: від 15 грн (Активоване вугілля) до 120 грн (Полівітаміни)
   - Додано 5 аптек з українськими адресами
   - Автоматично створюється inventory для кожної аптеки

2. **Список препаратів з цінами:**
   ```
   Парацетамол 500 мг       - 20.00 грн (OTC)
   Ібупрофен 200 мг         - 30.00 грн (OTC)
   Амоксицилін 500 мг       - 70.00 грн (Rx)
   Активоване вугілля       - 15.00 грн (OTC)
   Аскорбінова к-та         - 25.00 грн (OTC)
   Німесулід 100 мг         - 45.00 грн (Rx)
   Омепразол 20 мг          - 55.00 грн (Rx)
   Лоратадин 10 мг          - 35.00 грн (OTC)
   Полівітаміни             - 120.00 грн (OTC)
   Цефтріаксон 1 г          - 90.00 грн (Rx)
   ```

3. **Створено скрипт `reset-database.bat`:**
   - Автоматично видаляє стару БД
   - Створює нову з правильними даними
   - Один клік ? готово!


### Спосіб 2: Вручну через команди

```powershell
# 1. Видалити базу даних
dotnet ef database drop --force

# 2. Застосувати міграції
dotnet ef database update

# 3. Запустити додаток (seeder створить дані)
dotnet run
```

### Спосіб 3: SQL запит (якщо не хочете видаляти БД)

```sql
-- Оновити ціни препаратів
UPDATE Drugs SET Price = 20.00 WHERE Name LIKE '%Парацетамол%';
UPDATE Drugs SET Price = 30.00 WHERE Name LIKE '%Ібупрофен%';
UPDATE Drugs SET Price = 70.00 WHERE Name LIKE '%Амоксицилін%';
UPDATE Drugs SET Price = 15.00 WHERE Name LIKE '%Активоване%';
UPDATE Drugs SET Price = 25.00 WHERE Name LIKE '%Аскорбінова%';
UPDATE Drugs SET Price = 45.00 WHERE Name LIKE '%Німесулід%';
UPDATE Drugs SET Price = 55.00 WHERE Name LIKE '%Омепразол%';
UPDATE Drugs SET Price = 35.00 WHERE Name LIKE '%Лоратадин%';
UPDATE Drugs SET Price = 120.00 WHERE Name LIKE '%Полівітаміни%';
UPDATE Drugs SET Price = 90.00 WHERE Name LIKE '%Цефтріаксон%';

-- Оновити ціни в складських залишках
UPDATE InventoryItems SET UnitPrice = d.Price
FROM InventoryItems i
INNER JOIN Drugs d ON i.DrugId = d.Id;
```
